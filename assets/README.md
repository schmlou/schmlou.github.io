# Assets Folder

This folder contains images and icons for your portfolio website.

## Required Files:

### 1. **favicon.ico**
- Browser tab icon (16x16 or 32x32 pixels)
- Create one at: https://favicon.io/
- Or convert an image at: https://convertio.co/png-ico/

### 2. **profile-photo.jpg** (Recommended)
- Your professional photo for the About section
- Recommended size: 400x400 pixels
- Should be a clear headshot or professional photo

### 3. **Project Images** (Optional)
Replace these placeholder files with actual screenshots:
- `wildfire-project.jpg` - Screenshot of Wildfire Risk Assessment project
- `urban-heat.jpg` - Screenshot of Urban Heat Island project
- `land-cover.jpg` - Screenshot of Land Cover Classification project

Recommended size: 800x600 pixels or similar aspect ratio

---

## How to Add Your Own Files:

1. Replace the placeholder files in this folder with your actual images
2. Keep the same filenames, or update the references in your code:
   - Favicon: `index.html` line 15
   - Profile photo: `static/about.md`
   - Project images: `static/projects.json`

3. Make sure image files are optimized for web:
   - Use JPG for photos
   - Use PNG for graphics with transparency
   - Keep file sizes under 500KB for faster loading
